Description :

Actimaths est un fork de Pyromaths qui permet de créer des fiches d'activités mentales avec leurs corrigés au format LaTeX et PDF.

Remerciements :

  • Pyromaths : Pour la base de code Python et l'idée originale.
  • Chris Coyier : Pour le système d'onglets Organic Tabs
  • Imagex : Pour la police de caractère One Way Or Another
  • Alex Gorbatchev : Pour l'embellisseur de code SyntaxHighlighter
